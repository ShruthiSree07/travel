# CalcVerse

Every calculator, one app. Finance, health, math, and everyday calculators — built with Next.js, Tailwind CSS, and TypeScript, deployed on Vercel.

## Calculators

- **Finance** — Loan/EMI Calculator, Compound Interest
- **Health** — BMI Calculator, Calorie & BMR
- **Math & Units** — Unit Converter, Percentage Calculator
- **Everyday** — Age Calculator, Tip Splitter

## Run it

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000).

## Stack

Next.js 16 (App Router) · React 19 · Tailwind CSS v4 · TypeScript · lucide-react icons. No backend, no database — every calculation runs client-side.

## Deploy

```bash
vercel --prod
```
